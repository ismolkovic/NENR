/*
		UPUTE ZA KORIŠTENJE 
*/

Potrebno je prevesti program pomocu 
		make 

1/3 i 2/3 projekta pokreću se naredbom:
	./LAB1 <file>


za 3/3 potrebno prevesti s 
	make fuzzyControl

pozicionirati u direktorij simulator
	cd Simulator 

Simulator se pokreće naredbom:
	java -jar Simulator

Na kraju sve pobrijesemo s 
	make clean

Config.txt je postavljena za korištenje na linuxu, promjeniti prema potrebi.
